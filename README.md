# 📊 Modelagem e Otimização com PuLP

## 📝 Descrição
Projeto desenvolvido para a disciplina de Pesquisa Operacional que consiste na implementação de dois modelos matemáticos de otimização utilizando a biblioteca **PuLP** em Python. O principal objetivo do projeto é modelar e resolver problemas de otimização linear e combinatória aplicados a cenários reais, demonstrando o uso prático da programação matemática para a tomada de decisões.

Os exercícios foram retirados do livro:

> Goldbarg, M. C., & Luna, H. P. L. (2005). *Otimização combinatória e programação linear: modelos e algoritmos*. Elsevier.

---

## 📚 Conteúdo

- Modelos matemáticos para problemas de alocação e planejamento.
- Implementação das soluções usando PuLP.
- Análise dos resultados obtidos pelos modelos.

---

## 🛠️ Tecnologias Utilizadas

- Python;
- [PuLP](https://pypi.org/project/PuLP/) - Biblioteca para programação linear e inteira em Python.
